What is BlockRentals?

BlockRentals is a decentralized marketplace that uses blockchain technology for house rentals. It allows for users to create a listing for their rental property, as well as allowing potential tenants to contact the landlord to create a rental agreement A blockchain, originally block chain, is a growing list of records, called blocks, that are linked using cryptography. Each block contains a cryptographic hash of the previous block, a timestamp, and transaction data. By design, a blockchain is resistant to modification of the data.
